## What's inside

| Tool | What it does |
|------|-------------|
| [FocusDim](https://www.getpeakflow.pro/focusdim) | Dims everything except your active window |
| [QuickBoard](https://www.getpeakflow.pro/quickboard) | Clipboard manager with search and pinning |
| [SoundSplit](https://www.getpeakflow.pro/soundsplit) | Per-app volume mixer from the system tray |
| [ScreenSlap](https://www.getpeakflow.pro/screenslap) | Full-screen meeting alerts from Google Calendar |
| [MeetReady](https://www.getpeakflow.pro/meetready) | Camera, mic, and lighting check before meetings |
| [Liquid Focus](https://www.getpeakflow.pro/liquidfocus) | Pomodoro timer with webcam-based focus tracking |

## Install

1. Download `PeakFlow.Setup.x.x.x.exe` from the [latest release](https://github.com/inchwormz/peakflow-releases/releases/latest)
2. Run the installer — PeakFlow lives in your system tray
3. 14-day free trial, no credit card required

**Requirements:** Windows 10/11, 64-bit
